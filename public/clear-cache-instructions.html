Browser Cache Clearing Instructions

Quick Method (All Browsers)

Press Ctrl + Shift + R (Windows/Linux) or Cmd + Shift + R (Mac) to force refresh the page.

Chrome/Edge

  1. Press F12 to open Developer Tools
  2. Right-click on the refresh button
  3. Select "Empty Cache and Hard Reload"

Alternative Method

  1. Open the admin panel in an incognito/private window
  2. Navigate to /admin
  3. Check if the Course Management link appears

Verify Changes

After clearing cache, you should see:

  • Course Management link in the admin sidebar (with a "NEW" badge)
  • The link should be between "Click Heatmap" and "Contact Submissions"
